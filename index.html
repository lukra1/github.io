PL-300 Exam Prep

Everything to go from zero to certified, in order.

Passing the PL-300 exam earns you the Microsoft Certified: Power BI Data Analyst Associate title. Follow this self-paced Microsoft path from start to scheduling day.

  1. Start creating your learning profile
    learn.microsoft.com/en-us/training/support/learn-profile-manage
  2. Get familiar with the PL-300 learning syllabus
    learn.microsoft.com/en-us/training/courses/pl-300t00
  3. Power BI Desktop download
    learn.microsoft.com/.../desktop-get-the-desktop
  4. Power BI tutorial — connect data & build reports
    learn.microsoft.com/.../service-get-started
  5. Watch the official videos on the Microsoft YouTube channel
    youtube.com/playlist?list=PLahhVEj9XNTdg4FnhAo_OQ...
  6. Register for Dashboard in a Day — self-paced, 8-hour hands-on class
    learn.microsoft.com/.../dashboard-in-a-day
  7. Register for Microsoft Virtual Training Days
    learn.microsoft.com/en-us/trainingdays
  8. Read: "Your road to Power BI certification starts here"
    powerbi.microsoft.com/.../your-road-to-power-bi-certification-starts-here
  9. Exam Study Guide — everything you need to know to pass
    learn.microsoft.com/.../study-guides/pl-300
  10. PL-300 exam page
    learn.microsoft.com/.../data-analyst-associate
  11. Watch the "Get Certified" PL-300 Reactor Series (5-part livestream replay covering each exam domain)
    aka.ms/pbi10/cert50
    Regional series: AMER · APAC — both in English. No dedicated LATAM series exists yet, but Microsoft Reactor's site supports English, Español, and Português via the language switcher at the bottom of the page.
  12. Watch Exam Readiness Zone prep videos
    learn.microsoft.com/.../preparing-for-pl-300-prepare-the-data
  13. Review the online exam guide for the full exam structure breakdown
    learn.microsoft.com/.../data-analyst-associate
  14. Try the official Exam Sandbox to get familiar with the exam interface
    learn.microsoft.com/.../data-analyst-associate#exam-sandbox
  15. Test your knowledge with the official practice assessment
    learn.microsoft.com/.../data-analyst-associate#practice-for-the-exam
  16. Check exam discounts — save 50% with a voucher from the Reactor test initiative
    community.fabric.microsoft.com/.../campaign_form
  17. Check student discounts, if eligible
    learn.microsoft.com/en-us/credentials/student-discounts
  18. Schedule your exam with Pearson VUE
    learn.microsoft.com/.../data-analyst-associate#schedule-exam
  19. Join the community — meetups with guest speakers, training resources, book clubs, and mentorship
    No link provided

Badges & Certifications

What you carry away.

M
Tier 1 — Module Badges

Earned throughout every week

Finish a single module and earn a digital badge on your Microsoft Learn profile — dozens accumulate as you go.

T
Tier 2 — Path Trophies

Earned at the end of each module

Complete a whole learning path and earn a trophy. Both live under Achievements on your Learn profile and share directly to LinkedIn — free, no exam.
